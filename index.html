About This Site

This is the website for the lab session of STA 210, where lab materials will be posted. The site and all the lab materials are hosted by Github, which is a web-based Git repository hosting service. It will allow you to easily share and cooperate with others on your code. Github is a natural choice (in my view) for this lab session since practicing statistical coding skills is our goal. I would recommend you to open a Github account and learn how to Git. It is very likely that there will be no mandatory assignments for this lab session, but I will post some optional ones for you to practice. Feel free to publish your code to me via Gitbub, and I will be able to comment and help you improve it easily using the Github features! Hope we will fave a fun semester. Cheers!

Lab 1 : Basic R

Content: This lab will give a very brief introduction of R, including the data structures (scalar, vector, matrix, dataframe and list), functions and some basic programming (if-statement, for-loop). In the lab I will go through an R script that I wrote, which covers several basic R operations.

Preparation: You can install R here and install RStudio here. By Wikipedia, "RStudio is a free and open source integrated development environment (IDE) for R, a programming language for statistical computing and graphics." If have time, you can take a look at A (very) short introduction to R.

Lab 2 : Exploratory Data Analysis (EDA) and one-way ANOVA

Content: Please download this R script. Lab 2 will cover some basic techniques in exploratory data analysis (how to quickly summarize a dataset, how to visualize the data to facilitate the analysis, etc.) In this lab we will also learn the "aov" function which performs ANOVA.

Assignment (Optional): This assignment explores how is the statistical power of F test in ANOVA related to the sample size. To be specific, when the null hypothesis is incorrect, how often can the F test corretctly rejects the null is related to the sample size. But is it a positive or negative relation? This assignment should be able to give you some clue. Have fun if you have some spare time :)

Lab 3 : Quantile-Quantile plot and log transformation.

Content: Please download this R script before the lab. The previous script is a partial one and the one with my code is here. Lab 3 will explore more on the Quantile-Quantile plot (Q-Q plot) and variable transformation. I will spend the first 10 to 20 minutes discussing how the Q-Q plot works. The above R script is a small project exploring the relationship between distribution features and the Q-Q plot, and I will give you 30 minutes to work on the project by yourself. Instructions are provided in the script. In the rest of the class I will step by step walk you through the coding of this project and comment on the log transformation technique.
